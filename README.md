# Crewfinder Rigging OS

Field-first rigging workflow, safety math, and collaboration tooling for entertainment riggers inside the Crewfinder ecosystem.

## Purpose
Crewfinder Rigging OS is a mobile-first rigging toolset for live events and entertainment production.

It is designed to help riggers, companies, and production teams handle:
- fast rigging calculations
- field verification
- job-based rigging workflows
- company safety rules
- reporting and signoff
- future collaboration and proof-of-work history

## Positioning
This product is:
- a field-use rigging workflow system
- a safety-aware deterministic calculation layer
- a collaboration surface for company and freelancer workflows

This product is not:
- a CAD suite
- a structural engineering certification tool
- a replacement for qualified engineering review

## Core Principles
- field-first, not CAD-first
- explainable, not black box
- admin-driven, not hardcoded
- mobile-first, not desktop-dependent
- workflow-integrated, not isolated
- safe defaults and visible warnings
