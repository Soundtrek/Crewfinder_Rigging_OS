# AI Use Cases

## Purpose
Define practical, safe AI use cases inside Rigging OS.

---

## UC1 — Explain Warning
User taps a warning.

AI returns:
- simple explanation
- contributing factors
- reference to inputs

---

## UC2 — Suggest Safer Setup
User requests help.

AI returns:
- possible safer configurations
- explanation of impact

AI must not change system values.

---

## UC3 — Job Summary
User opens job.

AI returns:
- summary of current state
- missing verification items
- unresolved warnings

---

## UC4 — Override Context Summary
Before override, AI can summarize:
- what is being overridden
- risk context

---

## UC5 — Report Summary
AI may generate:
- plain-language report summary
- explanation of key risks

---

## Rule
All outputs must clearly reference the deterministic system outputs they are based on.
