# Product Vision

## Product Name
Crewfinder Rigging OS

## Product Definition
Crewfinder Rigging OS is a field-first rigging workflow and safety support system for entertainment riggers, technical production teams, and companies operating in live events.

It combines deterministic rigging calculations, job-based workflows, safety checks, verification records, collaboration, reporting, and future proof-of-work integration inside the Crewfinder ecosystem.

## What This Product Is
- a field-ready rigging assistant
- a deterministic calculation layer for common rigging math
- a workflow system for rigging jobs and point verification
- a safety file support system
- a company and freelancer collaboration surface
- a premium, daily-use professional tool inside Crewfinder

## What This Product Is Not
- not a CAD suite
- not a structural design package
- not an engineering signoff platform
- not a substitute for legal duties under safety law
- not a hidden black-box AI product

## Core Positioning
This tool should be positioned as a practical operating system for real rigging work in the entertainment and live events industry.

The product must stay focused on the work reality of:
- load-in
- rigging setup
- temporary structure planning context
- venue compliance preparation
- point verification
- signoff and reporting
- documented safety workflow

## User Groups
### Freelancer Riggers
Need fast calculations, saved jobs, checklists, safety support, and proof of work.

### Company Technical Teams
Need reusable templates, safety rule controls, shared job workspaces, reporting, and compliance support.

### Production Managers / Safety Leads
Need oversight, verification trails, exportable records, and clear responsibility chains.

### Admin / Platform Governance
Need feature control, plan gating, legal text versioning, usage visibility, and auditability.

## Primary Problem
Current rigging tools are fragmented. Some solve math. Some solve documentation. Some solve planning. Very few connect the actual working flow from field calculation to job record to safety file to signoff.

Crewfinder can solve this by building one workflow-oriented system that supports the actual rigging process rather than isolated calculations.

## Strategic Objective
Create a premium tool that:
- gets used during real work
- improves safety communication
- increases professional retention inside Crewfinder
- creates tool-driven reasons for freelancers and companies to remain active on the platform

## South African Context
The tool must support South African live event operations and documentation expectations, especially around:
- Occupational Health and Safety Act context
- SANS 10366 live event safety practices
- safety files
- competency records
- temporary structures
- inspection and verification workflow

This context must be treated as operational support, not legal overclaim.

## Product Outcome
When complete, Rigging OS should feel like:
- the fastest calculator in the field
- the clearest safety workflow on a live site
- the most useful rigging admin record for a show
- a strong premium Crewfinder differentiator

## Product Standard
Every feature must pass all of the following:
- useful on site
- fast on mobile
- understandable without training overload
- explainable and auditable
- safe in presentation
- integrated with Crewfinder systems
